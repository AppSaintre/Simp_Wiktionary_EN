 Simp_Wiktionary_EN

[Intro]
This is a json-based simplified English Wiktionary.
I downloaded and parsed 466,000 English words from Wiktionary,
using the word list provided in https://github.com/dwyl/english-words.

[Usage]
lookup.py prestigious
